What is EBX?

EBX is a native blockchain token currently being developed by Ryan X. Charles.

Anyone with a GPU can become an EBX miner by connecting to an EBX mine. As of now, the only existing mine is EarthBucks™, owned by Ryan X. Charles. EBX serves as the universal blockchain ticker that will connect all future EBX mines.

To make EBX transactions, you must connect to a mine. Currently, the EarthBucks™ mine is accessible via its website or through its tRPC client (GitHub link). The EarthBucks™ website includes a built-in EBX wallet available to all users.

The EarthBucks™ mine client is not fully open source, but it provides open endpoints for connection. Most miners use the EarthBucks™ website, while advanced miners use the tRPC client, which requires programming ability.

Although no independent EBX applications have been developed yet, it is theoretically possible to build an EBX payment app that connects to the EarthBucks™ mine using the tRPC client.

This forum is dedicated to all things EBX. Welcome!
